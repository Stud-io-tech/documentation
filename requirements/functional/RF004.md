# RF004

## Caso de Uso

Atualizar Endereço de Entrega no Perfil

## Descrição

O sistema deve permitir que usuários atualizem o endereço principal de entrega em seu perfil de conta.

## Atores

Clientes
 
## Dependências

[RF003](/requirements/functional/RF003.md) 

## Cardinalidades

[ USER ] ( 0..1 ) ------------ ( 1..1 ) [ ADDRESS ]

Um usuário pode possuir somente um ou nenhum endereço. 

## Tabela de Dados

| **Nome** | **Tipo** | **Valor Obrigatório** |**Propriedade** | 
|----------|----------|-----------------------|---------------|
| id | UUID | Sim | Gerado automaticamente; valor único |
| address_id | UUID | Não | Chave estrangeira da tabela de endereço | 
| name | VARCHAR(255) | Sim | - |
| email | VARCHAR(255) | Sim | Valor único |
| image | TEXT | Não | - |
| active | BOOL | Sim | Valor padrão: true |
| created_at | TIMESTAMP | Sim | Valor padrão: horário atual; não será possível alterar  |
| updated_at | TIMESTAMP | Não | Valor padrão: horário atual; será possível alterar  |
