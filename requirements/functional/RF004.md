# RF004

## Caso de Uso

Gerenciar Loja

## Descrição

O sistema deve permitir que os usuários gerenciem suas lojas. Nesse caso, o gerenciar seria:
- Registrar;
- Editar;
- Desativar;
- Reativar.

## Atores

Usuário
 
## Dependências

[RF002](/requirements/functional/RF002.md)

## Cardinalidades

[ STORE ] ( 1..1 ) ------------ ( 0..1 ) [ USER ]

Uma loja deve pertecer no mínimo um usuário e no máximo um usuário. Contudo, não é necessário que um usuário possua uma loja, porém se houver necessidade, o usuário deverá possuir no máximo uma loja.
