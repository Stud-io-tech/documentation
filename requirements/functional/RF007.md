# RF007

## Caso de Uso

Gerenciar Produtos

## Descrição

O sistema deve permitir que os usuários gerenciem produtos da sua loja. 
Nesse caso, o gerenciar seria:
- Registrar;
- Editar;
- Desativar;
- Reativar.

## Atores

Usuário
 
## Dependências

[RF004](/requirements/functional/RF004.md)

## Cardinalidades

[ PRODUCT ] ( 1..1 ) ------------ ( 0..N ) [ STORE ]

Um produto deve pertecer no mínimo a uma loja e no máximo a uma loja. Contudo, não é necessário que um loja possua um produto, porém se houver necessidade, a loja poderá possuir muitos produtos.
