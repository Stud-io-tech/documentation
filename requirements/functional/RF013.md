# RF013

## Caso de Uso

Gerenciar Pedidos

## Descrição

O sistema deve permitir que os usuários gerenciem pedidos.
Nesse caso, o gerenciar seria:
- Listar;
- Filtrar;
- Detalhar;
- Editar status;
- Copiar link de pagamento Pix;
- Copiar localização do endereço no mapa.

## Atores

Usuário
