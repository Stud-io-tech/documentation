# RF012

## Caso de Uso

Gerenciar Itens de Carrinho

## Descrição

O sistema deve permitir que os usuários gerenciem os itens de carrinho no carrinho de compras.
Nesse caso, o gerenciar seria:
- Editar;
- Apagar.

## Atores

Clientes e Vendedores
