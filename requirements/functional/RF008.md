# RF008

## Caso de Uso

Visualizar Produtos

## Descrição

O sistema deve permitir que os usuários visualizem produtos das lojas. 
Nesse caso, o visualizar inclui:
- Listar;
- Detalhar;
- Filtrar.

## Atores

Clientes e Vendedores
