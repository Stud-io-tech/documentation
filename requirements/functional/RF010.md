# RF010

## Caso de Uso

Compartilhar Produto

## Descrição

O sistema deve permitir que os usuários compartilhem produtos fora do aplicativo. 
Nesse caso, o compartilhar inclui:
- Copiar link;
- Enviar QrCode com título;
- Enviar link com imagem e texto.

## Atores

Clientes e Vendedores
