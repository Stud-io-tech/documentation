# RF009

## Caso de Uso

Compartilhar Loja

## Descrição

O sistema deve permitir que os usuários compartilhem lojas fora do aplicativo. Nesse caso, o compartilhar inclui:
- Copiar link;
- Enviar QrCode com título;
- Enviar link com imagem e texto.

## Atores

Clientes e Vendedores
