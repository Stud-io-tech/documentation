# RF009

## Caso de Uso

Adicionar Produto no Carrinho

## Descrição

O sistema deve permitir que os usuários adicionem produtos no carrinho de compras.

## Atores

Usuário
 
## Dependências

[RF007](/requirements/functional/RF007.md)

## Cardinalidades

[ CART_ITEM ] ( 1..1 ) ------------ ( 0..N ) [ PRODUCT ]

Um item de carrinho deve pertecer no mínimo a um produto e no máximo a um produto. Contudo, um produto não precisa de um item de carrinho para existir. Entretanto, caso haja necessidade, um mesmo produto poderá se relacionar com mais de um item de carrinho.

[ CART_ITEM ] ( 1..1 ) ------------ ( 0..N ) [ USER ]

Um item de carrinho deve pertecer no mínimo a um usuário e no máximo a um usuário. Contudo, um usuário não precisa de um item de carrinho para existir. Entretanto, caso haja necessidade, um mesmo usuário poderá criar mais de um item de carrinho.

[ CART_ITEM ] ( 0..1 ) ------------ ( 0..N ) [ ORDER ]

Um item de carrinho deve pertecer no mínimo a zero pedidos e no máximo a um pedido. Contudo, um pedido não precisa de um item de carrinho para existir. Entretanto, caso haja necessidade, um mesmo pedido poderá possuir mais de um item de carrinho.
