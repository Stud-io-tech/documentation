# RN006

## Descrição

O preenchimento automático dos dados de endereço via API do ViaCEP é obrigatório, caso esteja funcionando no momento. Além disso, a inclusão opcional dos campos de latitude e longitude deverá ser realizada via API do OpenStreetMap, caso esteja funcionando no momento.
