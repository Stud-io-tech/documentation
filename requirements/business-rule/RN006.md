# RN006

## Descrição

O sistema deve tentar o preenchimento automático dos campos de endereço via ViaCEP. Em caso de falha da API, o cliente deve preencher manualmente o endereço. A inclusão opcional de latitude e longitude via OpenStreetMap será tentada se a API estiver funcional.
